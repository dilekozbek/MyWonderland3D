## Introduction

Welcome to MyWonderland3D! This project showcases a basic 3D world construction created with Unity. In this world, you will find dynamic features such as snowfall, rain, day-night cycles, a beautiful lake, various trees, flowers, mushrooms, and lush greenery.

## Getting Started

To get started with MyWonderland3D, follow these steps:

1. Clone this repository to your local machine.
2. Open Unity Hub and select the cloned directory as your project.
3. Once the project is loaded in Unity, navigate to the main scene file and double-click to open it.
4. You can now explore the 3D world and interact with its elements.

## Features

- Realistic day-night cycles.
- Dynamic weather effects including snowfall and rain.
- Beautifully crafted lake surrounded by trees and flowers.
- Lush greenery and detailed environment objects.

## Tanıtım

MyWonderland3D'ye hoş geldiniz! Bu proje, Unity ile oluşturulmuş temel bir 3D dünya inşasını sergiliyor. Bu dünyada, kar yağışı, yağmur, gece-gündüz döngüleri, güzel bir göl, çeşitli ağaçlar, çiçekler, mantarlar ve yeşil çimler gibi dinamik özellikler bulacaksınız.

## Başlarken

MyWonderland3D ile başlamak için şu adımları izleyin:

1. Bu depoyu yerel makinenize klonlayın.
2. Unity Hub'u açın ve klonlanan dizini projeniz olarak seçin.
3. Unity'de proje yüklendikten sonra ana sahne dosyasına gidin ve çift tıklayarak açın.
4. Artık 3D dünyayı keşfedebilir ve öğeleriyle etkileşime geçebilirsiniz.

## Özellikler

- Gerçekçi gece-gündüz döngüleri.
- Kar yağışı ve yağmur gibi dinamik hava efektleri.
- Ağaçlar ve çiçeklerle çevrili güzel bir göl.
- Yeşil bitki örtüsü ve detaylı çevre nesneleri.
